# Scan Tiket Film

Aplikasi yang digunakan untuk menscan suatu tiket film dengan menggunakan qrcode yang mereka miliki.

# Teknologi
1. Java
2. Google Vision
3. Barcode Reader

# Fitur
1. Scan Qrcode 
2. Hasil scan berupa tiket
